## Installation

#### Arch

[AUR package](https://aur.archlinux.org/packages/cherry-font/)

#### X11

```shell
$ ./make.sh
$ mv *.pcf /path/to/cherry-font/out # example : ~/.fonts
$ mkfontdir /path/to/cherry-font/out
$ xset +fp /path/to/cherry-font/out
```
